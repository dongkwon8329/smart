 💰 스마트 적금 추천 플랫폼 (Smart Saving Platform)

> **사용자의 경제 상황과 저축 목적에 맞는 최적의 금융 상품을 쉽고 빠르게 찾을 수 있도록 돕는 AI 기반 원스톱 추천 플랫폼입니다.**

---

 1. 프로젝트 개요 (Introduction)
* **기획 배경**: 시중에 넘쳐나는 복잡한 적금 상품들 사이에서 사용자가 겪는 선택의 피로감을 해소하기 위해 시작되었습니다.
* **목표**: 단순한 리스트 나열을 넘어, 설문조사와 AI 분석을 결합하여 개인 맞춤형 추천 및 오프라인 접근성(지도)을 제공하는 원스톱 플랫폼 구축을 목표로 합니다.

 2. 핵심 기능 (Key Features)
* **성향 분석 설문 시스템**: 나이, 직업, 저축 목적, 선호 금리 유형 등을 다각도로 분석하여 개인화된 금융 프로필을 생성합니다.
* **AI 맞춤형 상품 추천**: 설문 데이터를 기반으로 금융 챗봇 AI가 실시간 상담사처럼 상품의 장단점을 분석하여 사용자에게 최적화된 상품을 제안합니다.
* **지도 기반 지점 안내**: KakaoMap API를 활용하여 추천 상품을 취급하는 은행의 오프라인 지점 위치를 사용자의 현재 위치나 특정 거점 중심으로 시각화하여 제공합니다.
* **금융 상품 상세 비교**: 복잡한 우대 금리 조건과 중도 해지 이율 등을 직관적인 UI로 제공하여 사용자의 의사결정을 돕습니다.
* **관리자 전용 대시보드**: 신규 금융 상품의 동적 등록 및 수정, 사용자 이용 통계 관리 기능을 통해 플랫폼 운영의 효율성을 확보했습니다.

 3. 기술 스택 (Tech Stack)
* **Language**: Java(JDK 17), Javascript
* **Framework**: Spring Boot, JPA(Hibernate)
* **Database**: MySQL, Oracle Database
* **API**: OpenAI API(ChatGPT), KakaoMap API
* **Tool**: Git/GitHub, IntelliJ IDEA

 4. 담당 역할 (My Role)
* **AI 맞춤형 추천 엔진 구현**: 사용자의 설문 데이터(나이, 직업, 저축 목표 등)를 ChatGPT API에 전달하고, 분석 결과와 DB 정보를 매칭하는 로직을 개발했습니다.
* **지도 기반 금융 서비스 구축**: KakaoMap API를 활용하여 용인시 내 은행 지점 위치 시각화 및 사용자 근거리 지점 확인 기능을 구현했습니다.
* **백엔드 아키텍처 설계**: Spring Boot를 기반으로 상품 상세 정보 제공, 설문 데이터 관리, 사용자 인증을 위한 REST API를 구축했습니다.
* **관리자 모드 개발**: 신규 적금 상품 등록, 사용자 통계 확인 및 설문 회원 관리 플랫폼 관리를 위한 전용 페이지를 구현했습니다.
